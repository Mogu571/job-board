Description

工作职责

1. 产品定义与指标化

  • 独立输出 PRD、MRD、硬件规格书 HRD、ID Brief 等核心产品文档
  • 将情感陪伴、智能飞行等产品愿景，拆解为续航、噪音、重量、响应延迟、识别准确率、安全距离等可量化验收指标
  • 管理功能优先级，在性能、成本、周期间做合理决策

2. 研发项目管理

  • 基于 V-Model/Stage-Gate，拆解 EVT/DVT/PVT/MP 各阶段任务与交付物
  • 禅道搭建任务体系、依赖关系与关键路径，按 BMAD 框架敏捷迭代
  • 制定阶段评审标准，主持 Gate 评审，协同结构/电子/嵌入式/算法/ID 团队排期，提前识别风险

3. 量产与供应链前置

  • 前期识别元器件、模具、认证（FCC/CE/SRRC/民航局无人机备案/UN38.3 等）风险
  • 搭建 BOM 成本模型，组织 DFM/DFA/DFT 评审
  • 对接 EMS、结构件、电池/电机等供应商，推进样品验证与批次一致性

4. 跨团队协同

  • 承接 CEO 产品思路，转化为技术可执行方案
  • 以文档、指标、流程推进落地，联合 ID 打磨外观与整机体验

任职要求

硬性要求

  • 5–10 年硬件产品经理/机器人产品负责人经验
  • 至少主导 1 款消费级智能硬件/机器人从 0 到量产出货全流程
  • 熟悉硬件研发流程，精通 EVT/DVT/PVT/MP 各阶段交付标准
  • 能与结构/电子/嵌入式/算法专业沟通，看懂公差、BOM、功耗、算力分配
  • 可独立输出 PRD、硬件规格书、里程碑计划、BOM 成本模型
  • 具备供应链前置风险把控与量产避坑经验

加分项

  • 飞行器/无人机/机器人/锂电精密穿戴量产经验
  • 大疆、石头、追觅、科沃斯、Insta360、Pico、Anker、小米生态链、宇树、云深处、智元、逐际动力等背景优先
  • 熟悉 BMAD 敏捷硬件开发，熟练使用禅道
  • 熟悉 FCC/CE/SRRC、民航无人机法规等出海认证
  • 具备英文工作能力

软性素质

  • 逻辑清晰，交付物可直接执行
  • 擅长用文档与指标推进，不靠强势 push 与会议驱动
  • 冷静协同，抗压能力强，团队氛围友好
  • 对消费级产品质感与体验有审美判断力

我们不找这样的人

  • 纯软件/App/SaaS 背景产品经理
  • 只写文档、无完整量产落地经验者
  • 靠 push、频繁开会推进工作者
  • 只会提感觉型需求，无法输出量化指标者
