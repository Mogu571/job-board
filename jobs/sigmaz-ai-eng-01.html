Description

薪资：40K–60K/月｜团队背景：海外 AI 教育科技公司 Generative UI 创始团队

岗位职责

  1. 参与公司核心 AI 平台的整体架构设计与落地，负责智能体（Agent）及其编排能力的研发，实现大语言模型与工具、数据源及上下文记忆的有效整合。
  2. 基于 Python 构建具备生产可用性的后端服务，并将相关应用稳定部署至云端，持续优化系统性能与效果。

任职要求

  1. 具备基于 LangChain、LangGraph 等框架开发智能体的实际经验，理解多智能体协作、工具调用与任务规划等核心机制。
  2. 熟悉智能体记忆与上下文管理方法，能够处理多轮、长上下文对话场景；对 RAG 方案有扎实理解，包括向量数据库、Embedding 策略及检索流程。
  3. 具备模型效果评估与数据闭环意识，能够搭建评估流程、反馈机制和基础数据管道。
  4. 精通 Python，熟悉异步编程与 FastAPI 等后端框架，具备将 AI 能力封装为可扩展接口的经验。
  5. 具备云端部署经验，能够在 AWS、GCP 或 Azure 等平台上完成服务部署与扩展。
  6. 具备以数据和实验为导向的工程习惯，能够通过日志、版本管理和实验对提示词与智能体行为进行持续优化。

加分项

  • 有复杂 RAG 系统或聊天型智能体的实际项目经验，能够支持复杂检索并保持长期状态。
  • 具备编码类智能体（Coding Agent）的开发经验。
  • 具备扎实的数据分析与实验基础；熟悉 vLLM、Ray 等高并发或高吞吐 AI 服务框架。
  • 具备复杂文档处理经验，如 OCR、PDF 结构化解析或表格抽取等。

你将获得

  • 参与核心产品和底层架构建设的机会，直接影响公司技术路线和产品形态。
  • 在小而专注的创始团队中与 AI、工程和产品成员紧密协作，拥有较高的技术决策参与度。
  • 接触真实业务场景下的大模型与智能体系统，积累可持续复用的工程经验。
  • 具备与个人能力和贡献相匹配的薪酬与长期发展空间。
